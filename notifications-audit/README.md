# Notifications & Audit Module (Python)

## Developed By
**Pasupuleti Narasimha**

## Project Overview
This module is part of a Banking Management System and is responsible for handling
automated notifications and audit logging. It ensures that all important system
activities are properly notified to users and securely logged for monitoring
and compliance purposes.

## Features
- Email notifications for transactions, login alerts, and system updates
- SMS notifications (simulated) for OTPs, low balance alerts, and EMI reminders
- Centralized audit logging of all critical actions
- Automatic error logging and tracking

## Technologies Used
- Python 3
- SMTP (Email Service)
- Python Logging Module
- Modular folder structure

## Module Usage
This module is used in:
- Transaction alert system
- User login activity tracking
- System audit and compliance logging
- Notification automation
