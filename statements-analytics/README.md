# Statements & Analytics Module

## Author
G Preethi

## Description
This module handles bank statement analysis and report generation.

Features:
- Calculate total credit
- Calculate total debit
- Compute current balance
- Generate simple transaction reports

## Folder Structure

- `reports/`
  - `reports.py` → Generates formatted bank statements
    - Lists all transactions
    - Shows total credit, total debit, and current balance

- `analytics/`
  - `analytics.py` → Performs basic transaction analysis
    - Number of credits and debits
    - Largest credit and debit
    - Average transaction amount
